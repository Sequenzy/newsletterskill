# Newsletter Skill

Static landing page and agent skill for newsletterskill.com. Works with Codex, Hermes, OpenClaw, Claude Code, Cowork, and other skill-aware agents.

Install shortcut:

```bash
npx skills add newsletterskill
npx skills add Sequenzy/newsletterskill --skill newsletterskill
```

Raw skill folder: `skills/newsletterskill`
